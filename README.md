# Universal Icon Hub

Plataforma para buscar e personalizar ícones de alta qualidade.

## Funcionalidades

- **Busca**: Encontre ícones em português ou inglês
- **285.000+ Ícones**: Acesso a múltiplas coleções open-source
- **Personalização**: Ajuste cor e tamanho
- **Download**: SVG ou PNG
- **Interface otimizada**: Virtualização e infinite scroll

## Tecnologias

- React 19
- Vite
- Tailwind CSS
- Lucide React
- Iconify API

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse `http://localhost:5173`

## Build

```bash
npm run build
```
